# Paper1 colleague delivery package

This package is based on the latest highlighted manuscript delivery and adds a result-subsection reproduction tree for the revised algorithm and revised experiments.

Key folders:

- 00_HIGHLIGHTED_MANUSCRIPT: highlighted manuscript PDFs for review.
- 00_FINAL_PDFs: final PDF copies from the latest delivery.
- 01_latex_source_single_column_review_version: single-column LaTeX source.
- 02_latex_source_published_style_journal_version: published-style LaTeX source.
- 03_non_experimental_images: non-experimental figure assets and editable sources.
- 04_experiment_tables_and_data: original table/figure-source copy from the latest delivery.
- 06_experiment_reproduction_by_result_subsection: code, datasets, results, and plotting scripts split by result subsection.

Important run ids:

- P1--P8 use multi_tower_repair2_full_20260612.
- P9 public GIS uses repair2 20260612 rows and 20260612 multi-region summaries.
- P10 uses airlab_energy_calibration_stop_batch_20260606.
- P11 uses repair_stress_repair2_20260612.

The package includes generated/proxy data and public-source inputs where used. Public GIS risk labels and candidate-stop attributes remain proxy-generated, and AirLab telemetry is energy-calibration evidence rather than utility inspection field logs.
